# UPI Fraud Analytics Dashboard

## 📊 Project Overview
This project focuses on analyzing UPI transactions to identify patterns, trends, and potential fraud risks using SQL and Power BI.

---

## 🚀 Key Features
- Transaction trend analysis over time
- Failure rate analysis across apps and banks
- Identification of high-risk users
- City-wise transaction distribution
- Interactive dashboard with filters and KPIs

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Power BI
- DAX

## 💡 Key Insights

- Failure rates vary across apps and banks
